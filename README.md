--------------- PyCrypto Build Instructions ---------------
   - Downlaod a fresh copy from: http://ftp.dlitz.net/pub/dlitz/crypto/pycrypto/pycrypto-2.6.tar.gz
   - Extract tar bundle
   - cd pycrypto-2.6
   - sudo ./configure
   - sudo python setup.py build
